<h2>Tuiles Graphique - Star-Books</h2>

Création d'une Tuile graphique pour le futur webdesign d'un site vendant des romans de science-fiction

* Nom du projet : Star-Books
* Auteur : Mélick HERMAN - 2017

<h3>Contexte</h3>
<p>L'objectif est de créer un site permettant l'achat de différents type de livre (roman, BD, etc ..) ayant tous un rapport avec la science-fiction. Les cibles sont des personnes agées de 16 à 60 ans, le site misant sur des ouvrages plutôt récents.
</p>

<h3>Périmétre</h3>
La tuile graphique utilise des couleurs comme le violet et le noir pour représenter l'aspect futuriste. Le site sera épuré et en flat design, pour lui donner un coté moderne et se démarquer des sites de ventes de livres qui sont plus traditionnels.
